# User Registration App

A simple user registration application built with React. This app allows users to register, displays a list of users with pagination, and includes a form toggle for adding new users.

## Features

- **User List**: Displays a paginated list of registered users.
- **Add User Form**: A form to add new users to the list, with the option to show or hide the form.
- **Pagination**: Navigate through user pages with easy-to-use pagination controls.
- **Responsive Design**: UI is responsive and works on various screen sizes.

## Installation

1. Clone the repository.
2. cd frontend
3. npm install
4. npm run dev
5. cd backend
6. npm install
7. nodemon server.js
